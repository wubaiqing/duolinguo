# 初识英语

## 第二部分

### 语法点

复数

> 在中午里，无论你是在说一件外套或者五件外套，**外套**这个词不会改变。  
> 在英语里，你通常要加上 `s` 来表示复数。

| 单数                  | 复数                     |
| --------------------- | ------------------------ |
| one hat 一顶帽子      | two hats 两顶帽子        |
| one coat 一件外套     | two coats 两件外套       |
| one daughter 一个女儿 | three daughters 三个女儿 |

### 重点语句

谈论义务及价格

```text
How much is this coat?

这件外套多少钱？
```

```text
it is ten dollars.

这是十美元。
```

```text
I need two black hats.

我需要两顶黑色帽子。
```

```text
This jacket is expensive!

这件夹克太贵了！
```

```text
I like this shirt.

我喜欢这件衬衫。
```

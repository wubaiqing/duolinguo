# 第六部分

## 语法点

### 数字

> 注意在 `third` 之后，一下的所有数字都已 `-th` 结尾。

`one` -> `first`
`two` -> `second`
`three` -> `third`
`four` -> `fourth`
`five` -> `fifth`
`six` -> `sixth`
`seven` -> `seventh`
`eight` -> `eighth`
`nine` -> `ninth`
`ten` -> `tenth`

> 另外，英语没有表示`万`的单词，你可以就说 `ten thousand`。

`1,000` -> `one thousand`
`10,000` -> `ten thousand`
`100,000` -> `one hundred thousand`

注意哦，跟中文不一样，逗号是每三位打一个，不是四位！

## 重点语句

### 用序数词表达顺序

```text
Is this your frist week here?

这是你在这里的第一周嘛？
```

```text
This is my second week here!

这是我在这里的第二周！
```

```text
Is your company in this builing?

你的公司在这座楼里吗？
```

```text
Yes this is our office building.

是的，这是我们的办公楼。
```

```text
Do you usually work from home?

你平时在家工作吗？
```

```text
Yes, we work from home every day.

是的，我们每天都在家工作。
```

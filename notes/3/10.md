# 第十部分

## 语法点

### 数字

> 注意在 `third` 之后，一下的所有数字都已 `-th` 结尾。

`one` -> `first`  
`two` -> `second`  
`three` -> `third`  
`four` -> `fourth`  
`five` -> `fifth`  
`six` -> `sixth`  
`seven` -> `seventh`  
`eight` -> `eighth`  
`nine` -> `ninth`  
`ten` -> `tenth`  

> 另外，英语没有表示`万`的单词，你可以就说 `ten thousand`。

`1,000` -> `one thousand`  
`10,000` -> `ten thousand`  
`100,000` -> `one hundred thousand`  

注意哦，跟中文不一样，逗号是每三位打一个，不是四位！


## 重点语句

### 谈论未来的事件

```text
When will you get married?

你什么时候结婚？
```

```text
We will get married next August.

我们明年八月会结婚。
```

```text
Where will you live next year?

你明年会住在哪里？
```

```text
My husband and I will move to France next year.

我和丈夫明年会搬到法国。
```

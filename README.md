# 多邻国英语学习笔记

## 项目简介

本项目包含了多邻国英语学习课程的笔记和重点内容，旨在帮助学习者更好地掌握英语基础知识。每个笔记文件对应一个特定的学习主题，并包含相关的语法点、重点语句和示例。

## 目录结构

- `notes/1/`: 初识英语
- `notes/2/`: 谈论学校

## 使用方法

1. 克隆本项目到本地：

```bash
git clone https://github.com/wubq-code/duolinguo.git
```

2. 浏览 `notes` 目录下的 Markdown 文件，查看各个主题的学习笔记。
